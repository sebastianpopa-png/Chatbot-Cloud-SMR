Bienvenido a Chatbot Cloud SMR

Esta página web utiliza servicios cloud como RapidAPI y GitHub Pages para ofrecer un chatbot capaz de responder dudas básicas de informática. Usa el menú superior para acceder al chat o conocer más sobre el proyecto.

Acerca del proyecto

Este chatbot ha sido desarrollado como práctica del módulo DASP en 1º SMR. Utiliza una API alojada en RapidAPI (SaaS / API as a Service) y se publica mediante GitHub Pages como hosting estático en la nube.
